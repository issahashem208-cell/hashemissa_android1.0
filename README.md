# 🚀 Hashim Android Platform

منصة احترافية لنشر تطبيقات Android، بتصميم عصري وأداء فائق.

## ✨ المميزات

- تصميم Minimal Modern مع Glassmorphism
- شاشة بداية سينمائية مع جسيمات متحركة
- بحث لحظي (Instant Search)
- وضع داكن/فاتح
- نافذة تحميل مع عداد تنازلي
- لوحة تحكم Admin متكاملة
- متجاوبة 100% مع جميع الشاشات
- تحسينات SEO وأداء عالي

## 🛠️ التقنيات المستخدمة

- HTML5 / CSS3 / JavaScript (Vanilla)
- Font Awesome للأيقونات
- Google Fonts (Inter)
- Glassmorphism و Micro-interactions
